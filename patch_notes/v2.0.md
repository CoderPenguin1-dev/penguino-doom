### v2.0.0
Initial release of Penguino Dooom

**Important Changes**
- HUDs are now saved in the `PENGHUD` lump due to incompatibilites with `NYANHUD`
- `always_show_total` argument of the `map_time` HUD component is now `show_total`
    - Never show total time (0), show total only when continuing from a previous level (1), always show total (2)

**Additions**
- "Invulnerability Effect" option under `Display` to change the sky colormap behavior
- `-nocoloredblood` parameter to temporarily disable the "Colored Blood" setting
- `show_author` argument for the `announce_message` HUD component
    - Doesn't show author (0), shows author when known (1), always show author message (2)
